Cross-Platform Data Sync

All platforms connect to the same cloud PostgreSQL database. Start an investigation on your Mac, review findings on Windows, and check updates from your Android device — all data stays in sync.

What's Included

6 OSINT Modules

Person search, email intel, phone lookup, domain recon, web scraping, and social media enumeration.

Case Management

Create, track, and close investigations. Add targets with full PII fields. Generate reports.

Smart Correlation

Automatic cross-referencing of findings across targets using exact and fuzzy matching.

Multi-Format Reports

Export investigations as CSV, HTML, JSON, or PDF. Professional formatting ready for clients.

CLI + Web Dashboard

Full command-line interface for power users. Rich web dashboard for visual workflows.

Secure by Design

Parameterized queries, environment-based config, rate limiting, and input validation throughout.
